# CryptoVoiceApp
A Swift iOS app that fetches and speaks real-time cryptocurrency prices using CoinAPI, built to practice API integration and text-to-speech.

## Built With
- Swift
- UIKit
- AVFoundation
- CoinAPI

To get started with the project, follow these steps:
